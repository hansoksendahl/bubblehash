BubbleHash

A distributed-hash-table implemeted using IndexedDB and WebRTC.

Try It! Learn More

Free your data!

BubbleHash frees developers from the inefficient and costly server-client network topology. Its designed to scale for any size website and any number of users.

Peer-to-Peer

BubbleHash uses the Chord algorithm to distribute data between connected peers.

It's free!

BubbleHash is released as open-source under an MIT user license. Which means that anyone can use it for whatever thay want.

Supported Browsers

BubbleHash uses the WebRTC and IndexedDB APIs two recent additions to the HTML5 specification. Since WebRTC is an emerging technology it is recommended that you use the Beta versions of any of the browsers below.

Chrome
Firefox
Opera
